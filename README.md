## Installation

To get started with Zebra Learn, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/zebra-learn.git
    ```
2. Navigate to the project directory:
    ```bash
    cd zebra-learn
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```
